 Toxicity Detection System

Intern ID: CTTS191

 Project Overview

The Toxicity Detection System is a Machine Learning-based web application that identifies whether a user comment is toxic or non-toxic.

 Features

* Toxic comment detection
* Machine Learning classification
* User-friendly Streamlit interface
* Real-time prediction
* Trained using Jigsaw Toxic Comment Dataset

 Technologies Used

* Python
* Streamlit
* Scikit-Learn
* Pandas
* NumPy
* Joblib

 Dataset

Jigsaw Toxic Comment Classification Challenge Dataset

How to Run

Install dependencies:

pip install -r requirements.txt

Run application:

streamlit run app.py

 Output

The system predicts whether the entered comment is Toxic or Non-Toxic.
